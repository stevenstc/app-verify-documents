# Modelo de Negocio - DocuChain

## 📊 Resumen Ejecutivo

DocuChain es una plataforma SaaS (Software as a Service) que ofrece verificación y certificación de documentos usando tecnología blockchain. El modelo se basa en suscripciones mensuales pagadas en criptomoneda BTT (BitTorrent Token).

## 💰 Estructura de Ingresos

### Suscripción Principal

**Precio**: 1000 BTT/mes (~$1-2 USD dependiendo del precio del BTT)

**Características incluidas**:
- Certificación ilimitada de documentos
- Almacenamiento de hash en blockchain BTTChain
- Panel de administración completo
- Historial de documentos certificados
- API de verificación
- Soporte por email

### Servicios Adicionales (Futuro)

1. **Plan Empresarial**: 10,000 BTT/mes
   - Múltiples usuarios
   - API privada
   - Branding personalizado
   - Soporte prioritario 24/7
   - SLA garantizado

2. **Certificación Premium**: 100 BTT por documento
   - Para usuarios sin suscripción
   - Certificación individual
   - Sin acceso al panel

3. **API Empresarial**: Según uso
   - Autenticación por API key
   - Integración con sistemas empresariales
   - Webhooks para notificaciones

## 📈 Proyección de Crecimiento

### Año 1

**Meta de usuarios**: 1,000 suscriptores

**Ingresos mensuales**: 
- 1,000 usuarios × 1000 BTT = 1,000,000 BTT/mes
- Aproximadamente $1,000-2,000 USD/mes

**Costos operacionales**:
- Servidor: $50/mes
- MongoDB Atlas: $30/mes
- Dominio y SSL: $15/mes
- Gas fees blockchain: ~$100/mes
- Marketing: $300/mes

## 📊 Métricas Clave (KPIs)

1. **MRR (Monthly Recurring Revenue)**: Ingresos recurrentes mensuales
2. **Churn Rate**: Tasa de cancelación de suscripciones
3. **CAC (Customer Acquisition Cost)**: Costo de adquisición por cliente
4. **LTV (Lifetime Value)**: Valor de vida del cliente
5. **Active Users**: Usuarios activos mensuales
6. **Documents Certified**: Documentos certificados por mes

## 🔄 Ciclo de Vida del Cliente
